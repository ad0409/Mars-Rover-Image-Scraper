This is a sample Python directory to fetch data from NASA's Mars Rover Photo API
Rover Name: Curiosity

Instructions:
1) Install all required packages listed in requirements.txt file
2) Run start.py file

Info:
Pictures are taken from different onboard cameras of the Curiosity Rover. Feel free to play around with different query
parameters to fetch data from different cams.

More info about APIs Photo Attributes:
https://github.com/chrisccerami/mars-photo-api

Created this repo for myself after reading this beautiful blogpost about Python&APIs:
https://realpython.com/python-api/#calling-your-first-api-using-python
